# javascript-lotto-precourse

## 기능 목록

## 로또 (Lotto)

- [x] 번호들을 전달받아 로또를 생성한다.
  - [x] 로또 번호의 개수가 6개가 아니라면 예외가 발생한다.
  - [x] 로또 번호에 중복된 숫자가 있으면 예외가 발생한다.

## 사용자 (User)

- [x] 구입 금액을 전달받아 사용자를 생성한다.
- [x] 구입 금액으로 로또를 구입한다.
  - [x] 구입 금액이 로또의 가격으로 나누어 떨어지지 않으면 예외가 발생한다.

## 당첨 로또 (WinningLotto)

- [x] 당첨 로또 번호를 저장한다.
  - [x] 당점 로또 번호의 개수가 6개가 아니라면 예외가 발생한다.
  - [x] 당첨 로또 번호에 중복된 숫자가 있으면 예외가 발생한다.
- [x] 보너스 로또 번호를 저장한다.
  - [x] 보너스 로또 번호가 당첨 로또 번호와 중복되면 예외가 발생한다.

## 당첨 확인 (Checker)

- [x] 사용자가 구매한 모든 로또의 당첨을 확인하고 당첨 통계를 반환한다.

## 계산기 (Calculator)

- [x] 당첨 통계에 따른 수익 금액을 계산하고 수익률을 반환한다.

## 입력 뷰 (InputView)

- [x] 구입 금액을 입력 받아 숫자로 변환 후 반환한다.
  - [x] 입력된 구입 금액이 양의 정수가 아니라면 예외가 발생한다.
- [x] 당첨 로또의 번호들을 입력 받아 숫자 배열로 변환 후 반환한다.
  - [x] 입력된 당첨 로또 번호 중 양의 정수가 아닌 번호가 있다면 예외가 발생한다.
  - [x] 입력된 당첨 로또 번호 중 로또 번호의 범위에 포함되지 않는 번호가 있다면 예외가 발생한다.
- [x] 당첨 로또의 보너스 번호를 입력 받아 변환 후 반환한다.
  - [x] 당첨 로또의 보너스 번호가 양의 정수가 아니라면 예외가 발생한다.
  - [x] 당첨 로또의 보너스 번호가 로또 번호의 범위에 포함되지 않는다면 예외가 발생한다.

## 출력 뷰 (OutputView)

- [x] 구매한 로또들을 받아 각 번호를 오름차순으로 정렬한 후, 형식에 맞게 출력한다.
- [x] 당첨 통계를 받아 형식에 맞게 출력한다.
- [x] 수익률을 받아 형식에 맞게 출력한다.

## 로또 컨트롤러 (LottoController)

- [x] 유저가 가진 모든 금액으로 로또를 구매한 뒤 당첨 통계를 반환한다.
- [x] 유저의 구입 금액과 당첨 통계를 비교하여 수익률을 반환한다.
